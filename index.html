git clone - path to repository
git status - Текущий статус локального репозитория
git add: path to file - добавить новый файл под контроль Git
git commit -am "added -- explaine what was added" Зафиксировать изменения
git push - Добавить файл с зафиксироваными изменениями на сервер
git pull - Забрать файл из сервера
gitk --all& - Команда для открытия графической консоли
git checkout key for commit version - вернутся на нужную версию обновления, ключ можно взять из графической версии гит(для буфера обмена нажми ctrl+insert)
